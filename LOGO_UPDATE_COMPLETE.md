## 🚀 كيفية المعاينة

### الطريقة 1: استخدام Live Server (VS Code)

1. افتح VS Code
2. انقر بزر الماوس الأيمن على `public/index.html`
3. اختر "Open with Live Server"
4. سيفتح المتصفح تلقائياً

### الطريقة 2: استخدام Python

```bash
# في مجلد public
cd public
python -m http.server 8000
```

ثم افتح: `http://localhost:8000`

### الطريقة 3: استخدام Node.js

```bash
# إذا كان لديك http-server مثبت
npx http-server public -p 8000
```

ثم افتح: `http://localhost:8000`
