## 📝 تعليمات التقاط الصور

### الأدوات الموصى بها:
- **Windows**: Snipping Tool أو Windows + Shift + S
- **Mac**: Cmd + Shift + 4
- **Linux**: Flameshot أو GNOME Screenshot
- **المتصفح**: أدوات المطور (F12) + Screenshot

### إرشادات:
1. استخدم وضع ملء الشاشة
2. تأكد من وضوح النصوص
3. احذف المعلومات الحساسة
4. استخدم بيانات تجريبية واضحة
5. احفظ الصور بصيغة PNG
6. اضغط الصور لتقليل الحجم (يفضل أقل من 500KB)

### بيانات تجريبية للاستخدام:
- **البريد الإلكتروني**: researcher@example.com
- **اسم المستخدم**: باحث تجريبي

- **عنوان البحث**: بحث تجريبي للتوثيق
