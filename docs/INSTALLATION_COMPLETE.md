## 🚀 الخطوات التالية

### 1. إنشاء ملف `.env.local`
أنشئ ملف `.env.local` في جذر المشروع:
```env
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```

### 2. تشغيل المشروع
```bash
npm run dev
```

### 3. فتح المتصفح
افتح [http://localhost:3000](http://localhost:3000)
