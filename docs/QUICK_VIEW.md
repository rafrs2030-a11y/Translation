## 🖥️ كيفية المعاينة

### الخطوة 1: افتح المشروع
```bash
cd public
```

### الخطوة 2: شغل سيرفر محلي
اختر واحدة:

**Python:**
```bash
python -m http.server 8000
```

**Node.js:**
```bash
npx http-server -p 8000
```

**VS Code Live Server:**
- انقر بزر الماوس الأيمن على `index.html`
- اختر "Open with Live Server"

### الخطوة 3: افتح المتصفح
```
http://localhost:8000
```
