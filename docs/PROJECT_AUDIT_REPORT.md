#### 3.2 Backend
- ✅ Route `/api/auth/forgot-password` في `server/routes/auth.js`
- ✅ Route `/api/auth/reset-password-direct` (إعادة تعيين مباشرة)
- ✅ Route `/api/auth/reset-password` (إعادة تعيين عبر الرابط)
- ✅ Integration مع Supabase Auth

**الكود:**
```javascript
// الطريقة 1: عبر البريد
POST /api/auth/forgot-password
Body: { email }

// الطريقة 2: مباشرة
POST /api/auth/reset-password-direct
Body: { email, national_id, new_password }

// الطريقة 3: عبر الرابط
POST /api/auth/reset-password
Body: { password }
```

#### 3.3 Stores
- ✅ `authStore.js` يحتوي على:
  - `requestPasswordReset(email)`
  - `resetPassword(newPassword)`
